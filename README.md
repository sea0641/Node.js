## 1. 노드 시작하기   
* 노드란?   
Node.js는 Chrome V8 JavaScript엔진으로 빌드된 JavaScript런타임입니다.   
Node.js는 **이벤트 기반, 논 블로킹 I/O 모델**을 사용해 가볍고 효율적입니다. (+싱글 스레드)   
Node.js의 패키지 생태계인 npm은 세계에서 가장 큰 오픈소스 라이브러리 생태계이기도 합니다.
